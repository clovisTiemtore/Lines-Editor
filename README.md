# Lines Editor – JavaFX

Lines Editor is a JavaFX application developed as part of the *Architecture des Interfaces Homme-Machine* module (ENSISA – S7, 2026).

## Description
The objective of this project is to design an interactive line editor with a focus on graphical user interfaces and user interaction principles.

The application allows users to create and manipulate lines through direct interaction with the graphical interface.

## Features
- Line creation
- Selection of lines
- Movement and repositioning
- Resizing
- Deletion
- Inspector panel for line properties

## Technologies
- Java
- JavaFX
- GUI / HMI architecture

## Project Structure
- `lines/` : Java source code of the application
- `README.md` : project documentation

## Objective
The goal of this project is to apply JavaFX and HMI architecture concepts to build an interactive graphical editor.
